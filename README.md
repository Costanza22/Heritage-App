# Heritage Mapper

Projeto fullstack para visualização de locais históricos em mapa interativo.

A aplicação permite consultar pontos históricos armazenados em um banco de dados PostgreSQL com PostGIS e exibi-los em um mapa utilizando React e Leaflet.

## Funcionalidades

- Exibição de mapa interativo
- Listagem de locais históricos com marcador no mapa
- Integração com banco de dados geoespacial (PostGIS)
- API REST para consulta dos dados

## Tecnologias utilizadas

### Backend
- Node.js
- Express
- PostgreSQL
- PostGIS

### Frontend
- React
- React Leaflet
- Leaflet
